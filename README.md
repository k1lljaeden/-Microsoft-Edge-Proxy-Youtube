# -Microsoft-Edge-Proxy-Youtube
Автоматический прокси-расширение для браузера Это простое расширение для браузера, которое автоматически активирует настроенный прокси-сервер при включении расширения. После активации оно начинает перенаправлять весь трафик через указанный прокси без необходимости ручного включения. 

Как установить
            
В браузере откройте страницу расширений:
edge://extensions/

Включите режим разработчика.
Загрузите распакованное расширение, выбрав папку проекта.(Загрузить Распакованное)
После установки активируйте расширение — оно начнет работать автоматически.

Как настроить

В файле background.js 
Вы можете менять IP и Port который вам нужен.
В скрипте уже есть 3 прокси: Индия, Россия, Сша
      
Использование

После активации расширение автоматически перенастраивает браузер на использование вашего прокси-сервера.
Для отключения — просто выключите расширение.
